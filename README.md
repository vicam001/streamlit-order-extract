# streamlit-chat
Chat with AI Agent using Streamlit



## Create Python environment

```bash
python3 -m venv venv
source venv/bin/activate
```
To exit the virtual env:
```bash
deactivate
```

## Streamlit
```bash
pip install streamlit
streamlit hello
streamlit run app.py
```
